## 1. 产品概述

超市进销存管理系统是一款面向中小型连锁超市与便利店的综合性业务管理平台，覆盖商品采购、销售收银、库存周转、供应商协同与经营分析五大核心环节。
- **目标用户**：超市店长、采购员、收银员、库管员、连锁经营管理者
- **核心价值**：通过数字化流程替代手工台账，实现进销存数据实时联动、库存预警自动触发、经营数据可视化决策，降低损耗、提升周转效率

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|---------|----------|
| 店长（管理员） | 后台预置 | 全模块访问、数据报表、用户管理、系统配置 |
| 采购员 | 管理员创建 | 供应商管理、采购订单、入库单、库存查询 |
| 收银员 | 管理员创建 | 销售收银、退货处理、当日销售查询 |
| 库管员 | 管理员创建 | 库存盘点、出入库操作、库存预警处理 |

### 2.2 功能模块
1. **经营驾驶舱**：关键指标卡、销售趋势图、库存预警、热销榜单、快捷入口
2. **商品管理**：商品分类、商品档案、条码管理、单位换算、价格策略
3. **采购管理**：供应商档案、采购订单、采购入库、采购退货
4. **销售管理**：收银开单、销售单据、退货处理、促销活动
5. **库存管理**：实时库存、库存盘点、出入库流水、库存预警、库龄分析
6. **经营分析**：销售报表、采购报表、利润分析、商品排行、库存周转

## 3. 核心流程

**采购入库流程**：库管发现低库存预警 → 采购员创建采购订单 → 选择供应商添加商品明细 → 店长审核订单 → 供应商发货 → 库管确认入库实收数量 → 系统自动增加库存并生成应付账款

**销售收银流程**：收银员扫码录入商品 → 系统实时计算金额 → 应用促销/会员折扣 → 选择支付方式 → 系统扣减库存并生成销售单 → 自动更新经营驾驶仓数据

**库存盘点流程**：库管创建盘点单 → 选择盘点范围 → 录入实盘数量 → 系统对比账面数量生成差异 → 审核差异处理 → 自动调整库存并记录流水

## 4. 用户界面设计

### 4.1 设计风格
- **主题方向**：现代工业风 + 鲜活市集元素，深墨绿主色搭配琥珀金与奶白点缀，营造专业且富有生命力的零售管理氛围
- **主色**：深墨绿 `#0F3D2E`（沉稳信任）/ 琥珀金 `#D4A24C`（暖意活力）
- **辅色**：奶白 `#F5F1E8`、墨灰 `#1A1F1C`、警示橙 `#E07A3F`、信息青 `#5BA8A0`
- **按钮风格**：圆角6px、扁平+微阴影、主按钮深绿填充、次按钮描边
- **字体**：标题使用 `Fraunces`（衬线感、有腔调），正文使用 `Plus Jakarta Sans`（清晰现代），数字使用 `JetBrains Mono`（等宽对齐）
- **布局**：左侧固定侧边栏（深绿背景）+ 顶部面包屑工具栏 + 主内容区（奶白底）
- **图标**：Lucide线性图标，1.5px描边
- **数据可视化**：自定义SVG图表，配色与主题统一

### 4.3 响应式
- 桌面优先（≥1280px）：完整侧边栏 + 多列布局
- 平板（768-1279px）：可折叠侧边栏 + 双列布局
- 移动端（<768px）：底部Tab导航 + 单列堆叠，收银页保持双栏（小屏优化为上下结构）
